# 一般疑问句

## 概述

+ 可以直接用 `yes` `no` 来回答

## 对话

+ 对话1

  ```text
  // 一般疑问句
  Is this the teachers' office?

  // 回答
  Yes, it is.
  ```

  ```text
  // 陈述句
  This is the teachers' office.
  ```

+ 对话2

  ```text
  // 一般疑问句
  Is this the library?
  No, it isn't. This is the library.
  ```

  ```text
  // 陈述句
  This is the library.
  ```
